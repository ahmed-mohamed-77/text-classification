[![Hugging Face Model](https://img.shields.io/badge/HuggingFace-Model-blue)](https://huggingface.co/Ahmed5134/hf_food_not_food_text_classifier_distilbert_base_uncased)

## 📌 Overview  
This project provides a **food vs. non-food text classifier** using a fine-tuned DistilBERT model. The model determines whether a given text is related to food or not.  

## 🚀 Model Details  
- **Model Name:** [`hf_food_not_food_text_classifier_distilbert_base_uncased`](https://huggingface.co/Ahmed5134/hf_food_not_food_text_classifier_distilbert_base_uncased)  
- **Architecture:** DistilBERT (fine-tuned for text classification)  
- **Task:** Binary classification – "Food" vs. "Not Food"  
- **Framework:** 🤗 Transformers (PyTorch)
